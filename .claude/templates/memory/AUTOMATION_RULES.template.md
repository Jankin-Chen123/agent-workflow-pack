# Automation Rules

## Active Rules

### Active Work Disclosure

Every user-facing response must identify the active Agent, what it is doing, and what skill/plugin capability is being used.

```text
当前 Agent：<agent-name>
当前工作：<what is being done>
使用能力：<skills/plugins and why>
当前产物：<file or artifact path, if any>
```
